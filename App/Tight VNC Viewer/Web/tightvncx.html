What is TightVNCX?

TightVNCX is a fully-featured TightVNC viewer implemented as an ActiveX control. Just like other ActiveX controls, it can be embedded in other applications or documents. As an example, the archive available for download includes a sample HTML page with embedded TightVNCX viewer.

TightVNCX is being developed by Vladimir Vologzhanin, the code was initially based on the TightVNC 1.2.4 Win32 viewer.

Download Test Version

Click on this link to download the test version of the TightVNCX viewer (110,047 bytes). The archive contains the viewer DLL, and sample HTML pages that can be used as a container for the ActiveX control.
