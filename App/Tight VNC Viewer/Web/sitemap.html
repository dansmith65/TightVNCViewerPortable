Below is an annotated list of all the pages on the TightVNC Web site.

Main Area

  • Main Page - a brief overview of the software and the site.
  • Download - links to download the latest stable and development versions for all supported platforms.
  • Get Support - hints on how to get support, where to send your questions, and where to search for answers.
  • Contribute - numerous ways to support the TightVNC project, and why you might like to contribute.
  • Contact Us - how to contact TightVNC developers.
  • People - people behind the TightVNC project.
  • Site Map - this page.

Information

  • Introduction - what TightVNC is, supported features, TightVNC and other VNC software.
  • Screenshots - images of TightVNC software in action.
  • Documentation - links to various pieces of documentation:
    • TightVNC for Windows: Installation and Getting Started
    • Win32 BUILDING.txt - how to build Win32 version from the sources.
    • Unix README - how to build and install the Unix version.
    • vncviewer(1) - Unix manual page for the vncviewer program.
    • vncserver(1) - Unix manual page for the vncserver script.
    • Xvnc(1) - Unix manual page for Xvnc server.
    • vncpasswd(1) - Unix manual page for the vncpasswd utility.
    • vncconnect(1) - Unix manual page for the vncconnect utility.
    • Java Viewer README - documentation on the TightVNC Java viewer.
  • FAQ - Frequently Asked Questions with answers (under construction).
  • Mailing Lists - instructions on using TightVNC mailing lists.
  • Change Logs - links to the documents tracking the history of changes in TightVNC software:
    • What's New in TightVNC - a brief overview of changes introduced in each TightVNC release.
    • Change Log, Win32 Version - detailed list of changes in the Win32 codebase, extracted from Subversion.
    • Change Log, Unix Version - detailed list of changes in the Unix codebase, extracted from the Subversion.
    • Change Log, Java Viewer - detailed list of changes in the Java viewer codebase, extracted from the Subversion.

Development Resources

  • Report Bugs - how to report bugs in TightVNC software.
  • Suggest Features - how to send new feature requests.

Link Collections

  • Other Sites - a list of the most important Web sites related to the TightVNC project.
  • Related Software - a collection of links to TightVNC-related software titles.
