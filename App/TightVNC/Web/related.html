Preface

Here you will find a collection of links to software titles that can be used in conjunction with TightVNC, extend its functionality in some ways, or provide additional useful services.

If you would like to contribute a software title to this collection, please send an e-mail message to the project maintainer. In the message body, include a link to the primary Web site of the software and a short but informative description (no more than 512 bytes), like in the list below. Also, you can send a logo image, in the GIF format with transparent background. The recommended sizes for logo images are 32x32 to 48x48 pixels.

Note: we do not guarantee that your submission will ever be included in this page, and if it is included, we may remove it at any moment.

Software

TitleDescription
TightProjector

TightProjector is a software that can transmit the screen of a partucular computer to other computers in the same local-area network. The data is transmitted continuously, in real time. TightProjector is a multicast variation on the VNC theme.
VNCRobot

VNCRobot is a free automated test tool based on a VNC client. Apart from standard client functions it provides its own proprietary scripting language and a set of useful features which enable you to program, verify and reproduce behavior of a remote computer and provide a comprehensive report with test data and screenshots.
Kaboodle

 Kaboodle is a freeware Internet application which provides:
  • Complete visualization of your LAN, updated in near-real time.
  • A "personal VPN" capability to connect Kaboodle-enabled LANs together.
  • Secure wrappers for other Internet applications (file transfer, TightVNC etc.).
SmartCode
VNC Manager

 VNC Manager is an easy to use administration tool which allows you to manage computers running VNC or Windows Terminal Servers. The list of features includes but is not limited to:
  • Can control VNC/RDP servers from the unified GUI
  • Can scan network to discover active VNC/RDP servers
  • Built-in SSH tunneling support for VNC/RDP sessions
  • Supports UltraVNC Repeater, HTTP/SOCKS5 proxy
  • Allows viewing VNC screens as thumbnails
  • VNC Deployment and Password Reset Wizards
  • Computer management and inventory features
Server-on-CD

 Server-on-CD is a server solution for small and medium working groups. The Internet router and file server boot directly from the CD. Installation and maintenance efforts are reduced to minimum. No special knowledge about information technology is needed. The extended version offers user management and a lot of further features.
OSXvnc

 OSXvnc is a robust, full-featured VNC server for Mac OS X that allows any VNC client to remotely view and/or control the Mac OS X machine. OSXvnc is supported by Redstone Software Inc, developers of Eggplant for cross-platform testing and automation.
FastPush Fastpush is a script that can remotely install WinVNC on to a computer running NT, 2000 or XP. You need admin rights to do this, but it will work over any reasonable network connection.
VNC Reflector VNC Reflector is a specialized VNC server which acts as a proxy sitting between real VNC server (host) and a number of VNC clients. VNC Reflector can switch between different host connections on the fly, supports reverse host connections and hosts with variable desktop geometries. It supports full-control and read-only client connections depending on which of two passwords was used to initiate a client session. VNC Reflector is a Unix-oriented software but it can be compiled under Windows using popular Cygwin tools.
xf4vnc A VNC implementation for XFree86 v4.x architecture. It enables VNC access to the local X display, and provides a "vncfb" driver which is a memory driver equivalent to the original Xvnc server.
VNCScan VNCScan is the original management console for VNC. It can scan a range of IP addresses for VNC computers, organize remote VNC-enabled computers into groups, check connectivity, and more.
Collaborative VNC Collaborative VNC (currently for Unix only) makes it easier for multiple people to share a remote desktop. Each member of a shared desktop gets a different colored mouse cursor, and can see everyone else's mouse cursors moving around the desktop at the same time. Multiple methods are provided for managing who has control of the desktop at any given time.
VNCTracker Database storage of IP addresses for on-the-fly creation of .vnc connection files. IPs are learned either via a PHP POST form or tac/grepped from an email log. Currently uses Apache, PHP4, MySQL, gnu-pop3d/courier-imap, tac, grep, gawk, sed, bash.
SharedAppVnc SharedAppVnc allows individual windows to be shared or replicated between remote computers. It extends the VNC protocol to allow window regions to be shared rather than the whole screen. The SharedAppVnc viewer puts each shared window in its own frame which can be resized or positioned independently of the size or position of windows on the server or the other clients.
IPv6-enabled
TigthVNC		 IPv6-enabled TigthVNC application. Both client and server have been ported to support IPv6 under Linux and Windows XP. The Java viewer has been ported too.
